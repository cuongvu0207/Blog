Đổi mật khẩu quản trị

Đăng nhập Google (thành viên)

Chỉ cần Client ID Tạo OAuth Client trên Google Cloud, copy Client ID vào đây. Authorized JavaScript origins: http://localhost:8080 và link blog public.

Khôi phục qua Gmail

Cấu hình gửi mã qua Gmail Chỉnh file config.json: ownerEmail (Gmail chủ blog) và smtp.appPassword (Mật khẩu ứng dụng Gmail). Trang quên mật khẩu sẽ gửi mã 6 số tới email này.
